Permutations with compose have the following properties.
For all permutations u, v and w
  1. (u · v) · w = u · (v · w)
  2. There exist an identity, Id, with Id · v = v · Id = v
  3. For every u, there exist u-1, with u · u-1 = Id

  2. There exist an identity, Id, with Id · v = v · Id = v
